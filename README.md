# CR-SCM-Porject
The Course Registration System is a web-based application that simplifies course enrollment. Students can browse courses, register, and manage schedules, while administrators handle course creation and student records. It reduces manual work, prevents errors, and improves efficiency.

## Project Description

A Course Registration System that enables students to browse available courses, register for classes, and manage their schedules efficiently. The system also provides administrative features for managing course offerings, student records, and enrollment limits. It is designed to streamline the registration process and reduce manual workload through a structured and user-friendly interface.

## README

### Course Registration System

This project is a software application developed to simplify the process of course enrollment in educational institutions. It supports both student and admin functionalities.

### Features

* Student registration and login
* Browse and search available courses
* Enroll in or drop courses
* View enrolled courses and schedule
* Admin panel to add/update/delete courses
* Manage student records and enrollment limits

### Technologies Used

  * Frontend: HTML, CSS, JavaScript
  * Backend: Python(Flask)
  * Database: MySQL / MongoDB

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/course-registration-system.git
   ```
2. Navigate to the project directory

   ```bash
   cd course-registration-system
   ```
3. Install dependencies

   ```bash
   npm install   # or pip install -r requirements.txt
   ```
4. Run the application

   ```bash
   npm start   # or python app.py
   ```

### Usage

* Students can log in to register for courses
* Admins can manage courses and users

# Course Registration System – Requirements

## Technologies
- Python
- Flask
- SQLite

---

## 1. Functional Requirements

The system shall:

1. Allow adding new students.
2. Display the list of students.
3. Display available courses.
4. Allow a student to register for a course.
5. Display all course registrations with student and course names.

---

## 2. Non-Functional Requirements

1. The system should be simple and user-friendly.
2. The system should store data permanently using a database.
3. The system should respond quickly to API requests.
4. The system should be easy to maintain and extend.

---

## 3. Software Requirements

| Software | Version             |
|----------|---------------------|
| Python   | 3.x                 |
| Flask    | Latest              |
| SQLite   | Default with Python |

---

## 4. Hardware Requirements

- Minimum 4GB RAM
- Any system capable of running Python
